# QR Code Generator

A minimal single-page QR code generator with real-time preview, because all other QR code generators sucks.

## Features

- Real-time QR code generation as you type
- Error correction level selector (Low, Medium, Quartile, High)
- Multiple visual styles: Square, Dot, Rounded, Diamond
- Dark themed UI
- No build step — runs directly in the browser

## Usage

Open `index.html` in a browser or serve it with any static file server.

## Dependencies

- [qrcode-generator](https://github.com/nickoala/nickoala.github.io) v1.4.4 (loaded via CDN)
- [Inter](https://rsms.me/inter/) font (loaded via Google Fonts)
